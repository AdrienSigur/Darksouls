# Dark Souls Rust: A Text-Based Rogue-lite

Welcome to **Lordran-Rust**, a terminal-based RPG inspired by the Dark Souls universe. This project is a tribute to the challenging mechanics of FromSoftware, built with the safety and performance of the **Rust** programming language.
